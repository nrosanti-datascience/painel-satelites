# 🛰️ Painel de Satélites

Este projeto foi desenvolvido como parte da disciplina de **Desenvolvimento Mobile**.

O objetivo é criar um painel de controle em **React Native** que simula o monitoramento de satélites, exibindo métricas e gráficos de forma visual, organizada e responsiva.

## Funcionalidades

- 📊 Gráfico de barras dinâmico mostrando os 12 satélites ativos;
- 🏷️ Legendas alinhadas abaixo de cada barra para identificar cada satélite;
- 📈 Cards de métricas exibindo totais de satélites, alertas e falhas;
- 📜 Scroll habilitado para permitir rolagem quando o conteúdo ultrapassa a tela;
- 📱 Layout responsivo e organizado para facilitar a leitura dos dados.

## Tecnologias Utilizadas

- **React Native** – Framework principal para desenvolvimento mobile;
- **Expo** – Utilizado para executar e testar o aplicativo de forma prática;
- **TypeScript** – Linguagem utilizada na implementação da aplicação;
- **Flexbox** – Responsável pela organização dos elementos na interface;
- **ScrollView** – Componente utilizado para permitir a rolagem da tela.

## Como Executar o Projeto

### Instalar as dependências

```bash
npm install
```

### Iniciar o projeto

```bash
npx expo start
```

Após iniciar o Expo, utilize o aplicativo **Expo Go** em um dispositivo móvel ou execute o projeto em um emulador configurado.
