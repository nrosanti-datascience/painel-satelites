# Painéis Espaciais

Este projeto apresenta um painel de controle em React Native para visualização de dados simulados de satélites.

## Funcionalidades
- Gráfico de barras dinâmico mostrando 12 satélites ativos.
- Legendas alinhadas abaixo de cada barra.
- Cards de métricas exibindo totais de satélites, alertas e falhas.
- Layout responsivo com rolagem habilitada.

## Observação
Os dados são simulados e servem apenas para fins acadêmicos.
